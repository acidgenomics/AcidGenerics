# AcidGenerics

S4 generic functions for [Acid Genomics][] packages.

## Installation

This is an [R][] package.

```r
install.packages(
    pkgs = "AcidGenerics",
    repos = c(
        "https://r.acidgenomics.com",
        getOption("repos")
    )
)
```

[acid genomics]: https://acidgenomics.com/
[r]: https://www.r-project.org/
